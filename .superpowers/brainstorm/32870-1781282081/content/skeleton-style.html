What should the loading skeletons look like?

Three approaches — all using your existing white/grey admin theme. Mockups show the dashboard page while data loads.

A — Shimmer Sweep

Horizontal gradient wave moves across each block. Content-shaped placeholder blocks match the real layout. Most polished, feels like a premium app.

B — Wave Stagger Pulse

Each card pulses at a slightly offset phase — creates a gentle ripple/wave across the grid. Softer and more organic than shimmer. Cards appear to "breathe" sequentially left to right.

C — Colour-tinted Shimmer

Same shimmer wave as A, but each stat card's skeleton uses its actual accent colour (violet, blue, amber, green). Feels like the content is "almost there". More expressive but slightly busier.
